Important FAQ

What is the Document Object Model(DOM)?

HTML DOM (Document Object Model) is a hierarchical representation of HTML documents. It defines the structure and properties of elements on a webpage, enabling JavaScript to dynamically access, manipulate, and update content, enhancing interactivity and functionality.

How do you select an element from the DOM?

There are various way to find elements in DOM. The easiest way to find an HTML element in the DOM, is by using the element id. Example: const element = document.getElementById('blog-btn'); Another way of finding element is by tag name. Example: const element = document.getElementsByTagName("p"); It finds all the elements with paragraph tag.Find elements by className. Example: const x = document.getElementsByClassName("intro");Find elements by Css Selectors. If you want to find all HTML elements that match a specified CSS selector (id, class names, types, attributes, values of attributes, etc), use the querySelectorAll() method. Example: const x = document.querySelectorAll("p.introName");

What is event Delegation in the context of DOM, Why is it useful?

Event Delegation is a pattern based upon the concept of Event Bubbling. It is an event-handling pattern that allows us to handle events at a higher level in the DOM tree other than the level where the event was first received.

Event Delegation is a useful pattern that allows us to write cleaner code, and create fewer event listeners with similar logic.
With event delegation, we can create fewer event listeners and perform similar events-based logic in one place. This makes it easier for us to add and remove elements without having to add new or remove existing event listeners.

How do you manipulate an element's attributes and style using DOM?

There are several way how we can manipulate element's attributes. First we have to retrieve an attribute value from an HTML. To do so we can use getAttribute() method. const element=getElementById('idName'); const value = element.getAttribute('attributeName'). Then set attribute by using setAttribute(). element.setAttribute('data-custom-attribute', 'new value'). Remove attribute using removeAttribute(). element.removeAttribute('data-custom-attribute')


We can access an element's style through the style property. const element = document.getElementById('idName'); const backGroundColor = element.style.backgroundColor. element.style.backgroundColor='blue'. It is always better to manipulate styles through CSS classes: element.classList.add=('hidden').
